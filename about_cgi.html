What can Python do for us?

First we will serve the contents of this folder using the built in http.server module.

To do this use the python -m http.server command.

Now say that we a have list of all students and their ages stored in a file in CSV file.
This information can change at any time, we want the current information from the file to be displayed in a table when a person visits the page.
When the user visits the page, we want Python to read the file for us and generate the HTML resopnse.

Have a look at the file here.

But we can't always just send the user to a text file.

  • We might want to format the data nicely.
  • We might have to query a database for the correct information.
  • We may have to look into multiple files to get information for one page.
  • We may want to do some security checks before sharing information, like having a log in system.

If we start the server with the command python -m http.server --cgi, the server is configured to look treat the files in the cgi-bin/ folder as executable CGI Python files.
